linear_Regression

Linear regression is a statistical predictive analysis method primarily used to study the linear relationship between independent variables (features) and a dependent variable (target). It assumes that the relationship between these variables can be represented by a straight line (for one-dimensional features) or a hyperplane (for multi-dimensional features). During the training process of a linear regression model, the algorithm determines the best fit of this line or hyperplane by minimizing the difference between predicted values and actual values (often using mean squared error as the loss function). Decision Tree Regression

tree regression is the application of decision tree algorithms in regression tasks. It differs from classification decision trees mainly in that the leaf nodes store continuous numerical values instead of category labels. In building a decision tree, the algorithm continuously splits the dataset into subsets based on different feature values until certain stopping conditions are met, such as reaching the maximum depth or the variance within the node's data falling below a certain threshold. For predictions on new samples, decision tree regression starts from the root node and traverses down the branches that satisfy the sample's features until it reaches a leaf node. The average value or weighted average stored in the leaf node then becomes the predicted value.
